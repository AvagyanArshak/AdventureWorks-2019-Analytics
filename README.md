AdventureWorks 2019 Sales Analytics
📊 Project Overview
Sales analytics dashboard built using AdventureWorks 2019 database, following a full end-to-end BI workflow — from SQL Server data import to Power BI dashboard design.
🛠️ Tools Used
* Microsoft SQL Server
* Power BI Desktop (Power Query, DAX)
📁 Files
* `AdventureWorks2019.sql` — SQL queries for data cleaning and transformation
* `ADW2019.pbix` — Power BI dashboard file
📌 Dashboard Pages
* Sales Overview — Total sales, budget comparison, sales by category and city
* Product Details — Monthly product sales breakdown with Top 10 filter
* Customer Details — Customer-level sales analysis
✅ What Was Done
* Imported SQL Server data into Power BI
* Cleaned and filtered data in Power Query
* Built a star schema data model
* Created DAX measures for sales analysis
* Designed 3 executive-level dashboard pages
